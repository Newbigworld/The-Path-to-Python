## 8、 字典（Dict）

- 创建字典

        Person = {'Name': 'Yan Liu', 'Age': 31}

- 基本操作
    - 索引

            print(Person['Name'])          # 返回'Yan Liu'

    - 键、值、键值对
            Person.keys()
            Person.values()
            Person.items()
    - 新增
            >>> Person.update([('Height', 1.8)])
            >>> Person
            {'Name': 'Yan Liu', 'Height': 1.8, 'Age': 31}
    - 删除
            >>> Person
            {'Name': 'Yan Liu', 'Height': 1.8, 'Age': 31}
            >>> Person.pop('Height')
            1.8
            >>> Person
            {'Name': 'Yan Liu', 'Age': 31}
            >>> Person.popitem()
            ('Name', 'Yan Liu')
            >>> Person
            {'Age': 31}
    - 长度
            >>> Person
            {'Age': 31}
            >>> len(Person)
            1

    - 循环

***迭代中常用函数： range()、  enumerate()***
========

        # enumerate()
        li = ['zhao','qian','sun','li']
        for key, value in enumerate(li):
            print(key, value)

        # range()
        li = ['zhao','qian','sun','li']
        for i in range(0,len(li)):
            print(li[i])
